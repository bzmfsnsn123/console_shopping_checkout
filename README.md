# console_shopping_checkout 便利店购物结算系统
Python 控制台入门练手项目，综合字典、循环菜单、异常捕获、格式化输出、数量统计等基础知识点，适合新手巩固Python基础语法。

## 项目功能
1. 商品展示：预设商品编号、名称、单价，控制台打印完整商品菜单
2. 选购加购：输入商品编号完成选购，重复选购自动累加对应商品数量
3. 购物车详情查看：清晰展示每款商品的名称、单价、购买数量、单品小计，自动统计总件数与合计总价，表格对齐排版美观
4. 结算结账：一键完成账单结算，结算后自动清空购物车
5. 输入容错机制：非数字输入、不存在的商品编号都会给出友好提示，程序不会崩溃闪退

## 运行环境
Python 3.x 标准环境，仅使用Python内置基础语法，无需安装任何第三方库，直接运行即可。

## 使用教程
1. 下载 `shop_check.py` 文件，使用PyCharm/IDLE等Python编辑器打开
2. 直接运行脚本，进入程序主菜单
3. 菜单功能说明：
   - 1：选购商品，输入对应商品编号加入购物车
   - 2：查看完整购物车清单，包含数量、单价、小计、总件数、总价
   - 3：执行结算结账，账单核算完成后清空购物车
   - 0：正常退出程序
4. 选购同一款商品会自动统计购买数量，不会重复罗列条目。

## 核心学习知识点
1. 字典Dict：用于存储商品编号与商品信息，键值对快速查询数据
2. 字典遍历 `items()`：同时获取键、值，循环打印商品菜单
3. 字典购物车：利用键记录商品ID、值记录购买数量，实现数量累加统计
4. `while True` 无限主循环：实现常驻菜单界面，搭配 `break` 完成程序退出
5. try-except 异常捕获：拦截非数字输入，完善程序健壮性
6. f-string 格式化对齐：`<`左对齐+固定字符宽度，实现表格规整排版
7. 基础算术运算：单价×数量计算单品小计，累加计算订单总价
